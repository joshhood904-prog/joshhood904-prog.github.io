## How to Update

### Add or Remove Products
1. Open `index.html`
2. Find the "Featured Products" section
3. Add or remove product cards following the existing format

### Customize Colors
Edit `styles.css` to change the color scheme. Current colors:
- Brown (#8b4513)
- Tan (#d4a574)
- Light Beige (#f9f7f4)
